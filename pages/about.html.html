About This Project


This website was developed as part of an academic activity under the CC 104 subject, handled by the Head Department of Information Technology. The project serves as a practical requirement designed to strengthen foundational knowledge and technical skills in web development using HTML.

Structured Design Layout planning & flow
Clean Code Scalability & maintenance
Professional Standards Real-world documentation

Educational Objectives

The primary goal is to provide students with early exposure to structured web design and basic layout planning. By working on this project, students gain experience in creating functional websites aligned with standards used in both academic and professional environments.

The development of this website also prepares students for upcoming academic requirements, particularly for the midterm period. Early practice allows students to adapt more easily to advanced lessons involving styling, page linking, and content organization.

Project Concept

The website is structured to simulate a simple Barangay Information System. This concept encourages students to apply technical skills to community-based scenarios, demonstrating how informational content may be presented clearly to the public.

Academic Disclaimer: This project is intended strictly for educational purposes. The website does not represent an official government platform and is not intended for public service deployment.
